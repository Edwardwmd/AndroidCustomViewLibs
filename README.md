

此库作为Android自定义控件学习的分享，里面涉及多种控件，组件的自定义，当开发中无法使用某些控件时，我们就需要实现控件的自定义，完善类似功能，谨此，此库拿出来分享给大家参考学习，有问题可在issues提出，我会尽力与大家探讨解答。
